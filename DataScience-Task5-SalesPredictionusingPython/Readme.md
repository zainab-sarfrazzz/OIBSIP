# Sales Prediction Using Python

## Overview

This project uses Machine Learning to predict product sales based on advertising expenditure across different media channels such as TV, Radio, and Newspaper. The project includes data preprocessing, exploratory data analysis (EDA), model building, and performance evaluation.

## Objective

To build a regression model that predicts product sales using advertising spending data.

## Dataset

* Dataset: Advertising Dataset
* Features:

  * TV Advertising Budget
  * Radio Advertising Budget
  * Newspaper Advertising Budget
* Target:

  * Sales

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## Project Workflow

1. Imported required libraries.
2. Loaded and explored the dataset.
3. Checked for missing and duplicate values.
4. Performed Exploratory Data Analysis (EDA).
5. Created scatter plots and a correlation heatmap.
6. Split the dataset into training and testing sets.
7. Trained two regression models:

   * Linear Regression
   * Random Forest Regressor
8. Evaluated the models using:

   * Mean Absolute Error (MAE)
   * Root Mean Squared Error (RMSE)
   * R² Score
9. Visualized feature importance and residuals.
10. Compared model performance and selected the best model.

## Results

* The trained models were able to predict product sales from advertising budgets.
* TV advertising showed the strongest influence on sales.
* Random Forest Regressor achieved better performance than Linear Regression (based on the evaluation metrics).

## Conclusion

This project demonstrates the complete workflow of a regression-based machine learning problem, from data preprocessing and visualization to model training, evaluation, and interpretation.
